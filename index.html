About

Versão simplificada do jogo "Blackjack", também conhecido como "21". Esse projeto foi desenvolvido com a intenção de experimentar com Programação Orientada a Objetos e revisar conceitos aprendidos ao longo do curso extracurricular de JavaScript que estou fazendo. O projeto ainda está em desenvolvimento, sendo meu próximo passo adicionar elementos gráficos ao jogo, como cartas e animações.

Jogabilidade

  • O objetivo do jogo é derrotar o Dealer.
  • O ganhador da rodada é aquele que possuir a maior quantidade de pontos na mão, mas a pontuação não pode passar de 21.
  • Aquele que ultrapassar a pontuação de 21 perde instaneamente.
  • O botão "Call" serve para pescar cartas.
  • O botão "Hold" é utilizado para segurar as suas cartas e passar a vez para o Dealer pescar.
  • O botão "Reset" deve ser usado para resetar as pontuações e reinicar a rodada.
